2. Nano Icon (Smallest Size)
────────────────────────────
Used for the smallest Ledger Manager / device-rendered icon.

Files:

- Geometry is intentionally simplified and thickened.
- This is standard practice for Ledger hardware UI.
- Do not replace with a scaled-down version of larger icons.
